# Style System

## 固定风格

角色母句：

```text
3渲2厚涂赛璐璐运动动画，3D体块基础结合2D手绘笔触，大色块设计，整块整块的平涂色面，硬朗图形化明暗切面，粗粝干刷笔触，轻微颜料颗粒感，避免碎乱AI噪点，不是写实真人摄影，不是光滑CGI，不要塑料玩具质感
```

场景母句：

```text
粗粝厚涂三渲二动画，可信三维空间结合二维手绘表面，强块面光影，边缘粗糙，低饱和电影色彩，清晰剪影，哑光磨损材质，前中后景分层
```

## 人物风格

- 默认白底/浅灰底全身人物概念设定图；三视图只在用户明确要求时使用。
- 比例：严肃动画式夸张，小头、长腿、窄肩或宽肩、大手大脚，不Q版。
- 表情：五官简化，骨相/眼窝/脸部色块表达气质，避免偶像脸和真人摄影。
- 服装：2-4 个大色块；材质用粗布、旧皮革、毛毡、短毛皮、暗哑金属，但不要材料清单过长。
- 颜色：1 个主色块 + 1-2 个辅助色块 + 1 个暖色钩子。想要冲击力时优先橘色编绳、橘色风带、橘色符结、橘色火铃。
- 动物/怪物头：写成面具化、雕塑化、几何化，大色块概括；禁止真实动物头、真实毛发、湿眼、毛孔、碎羽毛。

## 场景风格

- 先锁视觉中心，再写前景、中景、背景。
- 建筑/自然物用大形概括，生活痕迹清楚但细节克制。
- 统一光照方向和色温；冷灰蓝/灰褐/炭黑为底，暖橘只作局部情绪色。
- 避免霓虹、过锐、杂乱、真实品牌、可读文字、字幕、Logo、UI。
